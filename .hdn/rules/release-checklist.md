# HDN release checklist

A website change is not complete until all items pass.

- Build completes without errors.
- Required pages exist and are not unexpectedly small.
- JP/EN links work in both directions.
- Desktop header spacing is visually checked.
- Mobile navigation is usable without overlap.
- CTA text meets contrast requirements.
- Production URL is checked after deployment.
- Desktop and mobile screenshots are retained for the issue.

Do not close the issue before production verification.
